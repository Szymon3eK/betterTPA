# betterTPA
Najlepszy plugin na serwer SMP do teleportacji!

# Permisje
brak

# Komendy

<ul>
  <li>/tpa - otwiera gui </li>
  <li>/tpaccept <nick> - akceptuje zapytanie o teleportacje</li>
  <li>/tpdeny <nick> - odrzuca zapytanie o teleportacje</li>
</ul>
